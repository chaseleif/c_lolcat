# lolcat
___

## building

run `make`

you will get an output file `lolcat`, put this somewhere in your `$PATH`.
___

## what this is

this is a `C` rendition of the popular program `lolcat`.

the resource usage will be very low
and the runtime will be very fast.
___
## this is "unsafe"

as-of-yet:
- I don't do much safety checking
- I don't account for non-ASCII or whatnot

## this is safe for
plain ASCII files
